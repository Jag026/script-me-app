# script-me-app

This application is a sales tool that allows businesses to delevop sales scripts for their sales team. Users can create new scripts or clone an existing script template. 

To view the live project click [here](https://script-me-beta.herokuapp.com/)

Technologies used: JavaScript, Node.js, Express.js, PSQL, Pug.js, CSS, Heroku

The main features of this app are:

- User self registration - users can create an account and securely store their information and scripts. 
- Creating a new script - any logged in user can create a new script. Each script is made of parts (see screenshot below) and each part has a response.
- Cloning a script - any logged in user can clone a replica of an existing script and make edits to the new script
- Delete scripts - any logged in user can delete a script
- Edit scripts - any logged in user can edit scripts/parts/responses
